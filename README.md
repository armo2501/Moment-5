# Labbgrund till Moment 5 i kursen DT084G, Introduktion till programmering i JavaScript
Använd dessa filer som grund till din laboration.
Webbapplikationen visar Sveriges Radios kanaler och deras programtablåer via SR:s öppna API.
Användaren kan välja antal kanaler, visa tablåer, samt spela upp live-radio direkt i webbläsaren.
Byggd med JavaScript, Fetch API och DOM-manipulation.
